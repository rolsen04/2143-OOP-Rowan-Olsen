## What is OOP?

Object-Oriented Programming is a programming paradigm built around **objects** — self-contained units that bundle data (attributes) and behavior (methods).

Instead of writing code as a sequence of instructions, you model your program as a collection of interacting objects, each responsible for its own state and actions.

### Why OOP?

- **Modularity** — Break complex problems into smaller, manageable pieces
- **Reusability** — Classes can be reused across programs
- **Maintainability** — Objects are easier to debug and extend
- **Abstraction** — Hide complexity behind clean interfaces

---

## Core Principles

1. **Encapsulation** — Bundle data with the methods that operate on it
2. **Inheritance** — Create new classes based on existing ones
3. **Polymorphism** — One interface, many implementations
4. **Abstraction** — Show only what is necessary, hide the rest
